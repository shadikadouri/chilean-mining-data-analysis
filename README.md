# 🇨🇱 Chilean Copper Production & Trade Analysis

## 🎯 Project Overview
This project explores the dynamics of the Chilean mining sector, with a specific focus on the production and trade exports of copper. Using official data from Cochilco (Chilean Copper Commission), the analysis aims to extract key business insights regarding market trends, production volumes, and economic impact. 

This repository serves as a bridge between raw data processing and business strategy reporting.

## Tools
* **Programming Language:** Python
* **Environment:** Google Colab
* **Data Manipulation:** Pandas, NumPy
* **Data Visualization:** Matplotlib, Seaborn
* **Reporting:** Business Presentation (PDF)

## Repository Structure
* `analisi_dati.ipynb`: The Jupyter Notebook containing the data cleaning process, exploratory data analysis (EDA), and code for data visualization.
* `CHILEAN_PRODUCTION_AND_TRADE_EXPORTS_OF_COPPER.pdf`: The final slide deck summarizing the key findings for business stakeholders.
* `dati_cochilco.csv`: The dataset used for the analysis (sourced from Cochilco).

*Project developed as part of the MSc in Data Analytics for Business and Society @ Ca' Foscari University.*
